# Dragon News

Modern news portal built with Next.js.

## Features
- **Auth**: Secure login and registration.
- **News**: Dynamic news listing and detailed views.
- **UI**: Responsive design with Navbar and Header components.
- **Fonts**: Custom typography (Poppins & Montserrat).

## Tech
- Next.js (App Router)
- Tailwind CSS

## Run
```bash
npm install
npm run dev
